This project is a multi-threaded Java application that simulates a supermarket environment using concurrent programming techniques like Semaphores, ReentrantLocks, and ScheduledExecutors. It features a dynamic GUI (Swing) where users can configure simulation parameters, monitor customer flow in real-time, and manage checkout lane behavior, including cashier breaks and queue management.
